== Description ==

This plugin let you add via Customs Fields box Google-specific metatag to help determine how to best classify your content..

== Installation ==

1. Upload `gn-news-keyword.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place in the custom field box `gn-keyword-meta-tag` this only one time. After you'll find the value in the dropdown list.

== Frequently Asked Questions ==

= Do I need to use this plugin if my site isn't listed in Google News? =

No, you don't.
